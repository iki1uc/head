# HEAD – D1 · iki1uc

## Zweck
HEAD ist die Bewusstseins-Ebene (D1) des 6E-Würfelsystems.
Es steuert Logik, Kontrolle und die gesamte Ausrichtung des Systems.
HEAD ist die zentrale Achse für Cluster, Gates, Wurmloch-Systeme
und ALL-Multiplikationen.

## Ebenen
- D0 = style  
- D1 = HEAD  
- D2 = HTML  
- D3 = body  
- D4 = ANKER (Master)  
- D5 = API (Slave)

## TMP-Läufer
SYQ verbindet D5 mit META und stabilisiert Übergänge.

## Parität
D4 ↔ D5 bilden die Master/Slave-Achse.

## Regeln
- Exakt 6 Ebenen (Würfel-Gesetz)
- D1 ist die Bewusstseins-Ebene
- META ist Beobachter, nicht Ebene
