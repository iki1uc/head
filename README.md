HEAD ist die Bewusstseins‑Ebene (D1) des 6E‑Würfelsystems.  
Sie steuert Logik, Kontrolle, Ausrichtung und Priorität.  
HEAD ist die zentrale Achse für:

- Cluster‑Navigation  
- Gate‑Verhalten  
- Wurmloch‑Systeme  
- ALL‑Multiplikationen  
- 12e‑Stabilität in jede Richtung  

HEAD ist die Ebene, die entscheidet, wie das System denkt, reagiert und sich ausrichtet.

## Ebenen (6E‑Würfelgesetz)

- **D0 = style**

- jeder Übergang ist **12‑fach geprüft**  
- HEAD führt **12‑fach‑Checks** durch  
- System bleibt stabil, egal was passiert
